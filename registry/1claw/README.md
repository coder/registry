# 1Claw

This namespace hosts modules that connect [Coder](https://coder.com) workspaces to [1Claw](https://1claw.xyz) for vault-backed secrets and MCP tooling.

## Modules

- **oneclaw**: Provisions 1Claw vault and agent identities and writes MCP config for Cursor, Claude Code, and other MCP clients.
